# Greeting Servlet Application (Without Login)

A simple Java web application that generates personalized greeting messages using **Java Servlets**, **JSP**, and the **Spring Framework**. This project demonstrates the MVC architecture without implementing user authentication.

---

## Project Overview

This application allows users to enter their name and receive a personalized greeting message. It is designed to demonstrate the integration of Servlets, JSP, and Spring Framework while following the MVC design pattern.

---

## Features

- Personalized Greeting Message
- Spring Dependency Injection
- MVC Architecture
- Servlet-Based Request Handling
- JSP Frontend
- Simple and User-Friendly Interface

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/ArpitGupta0212/BridgeLabz_backend_Training.git
```

2. Switch to the project branch.

```bash
git checkout Greeting_Servlet_Application1
```

3. Import the project into IntelliJ IDEA.

4. Configure Apache Tomcat.

5. Deploy the WAR artifact.

6. Start the server and open the application in your browser.
